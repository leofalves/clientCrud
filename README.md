# clientCrud

**Final Project for module one in DS Bootcamp**

- Develop a CRUD REST API for client entity
-  Find all (with pagination)
-  Find by Id
-  Insert new (POST)
-  Update (PUT)
-  Delete (DELETE)

*Using Java 11 and Spring Boot 2.6.6*

Maven Dependencies
- Spring Web
- Spring JPA
- H2 Database
